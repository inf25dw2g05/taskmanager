**Capítulo 1 — Introducción**  
**1.1. Descripción del proyecto**  
**TaskManager Web Application** es una aplicación REST desarrollada en el marco de la asignatura *Desarrollo Web II* (M2) del grado en Informática de la UMAIA, curso 2025/2026.  
El proyecto reutiliza la API REST desarrollada en la primera evaluación continua y añade una aplicación cliente desarrollada con ReactJS para consumir los recursos ofrecidos por dicha API y proporcionar una interfaz gráfica para la gestión de proyectos y tareas..  
El sistema permite a los usuarios autenticados gestionar proyectos personales y las tareas asociadas a esos proyectos mediante una interfaz web. Cada usuario dispone de un espacio aislado: únicamente puede acceder, crear, modificar y eliminar sus propios proyectos y tareas. Las credenciales y la autorización continúan gestionándose mediante tokens JWT.  
La solución está compuesta por una base de datos MySQL, una API REST desarrollada con Node.js y Express, y una aplicación frontend desarrollada con ReactJS. Los tres componentes se despliegan mediante Docker Compose.  
**1.2. Objetivos**  
El proyecto cumple con los requisitos obligatorios del enunciado:  
- Implementar una arquitectura basada en servicios REST utilizando los verbos HTTP GET, POST, PUT y DELETE.  
- Gestionar tres recursos principales: users, projects y tasks.  
- Mantener las relaciones 1 entre usuarios y proyectos, y entre proyectos y tareas.  
- Utilizar representaciones de datos en formato JSON.  
- Implementar mecanismos de autenticación y autorización mediante JWT.  
- Garantizar el aislamiento de los recursos por propietario.  
- Desarrollar una interfaz web utilizando ReactJS.  
- Permitir la consulta y modificación de información mediante operaciones CRUD desde la aplicación cliente.  
- Gestionar la sesión del usuario autenticado desde el frontend.  
- Implementar una navegación simple entre las diferentes secciones de la aplicación.  
- Desplegar la solución mediante Docker Compose utilizando contenedores independientes para la base de datos, la API y el frontend.  
   
   
   
   
**1.3. Stack tecnológico**  
| | |  
|-|-|  
| **Capa** | **Tecnología** |   
| Frontend | ReactJS |   
| Framework web | Express 5 |   
| Base de datos | MySQL 8.0 |   
| Driver MySQL | mysql2 (con pool de conexiones) |   
| Autenticación | JSON Web Tokens (jsonwebtoken) |   
| Hash de contraseñas | bcryptjs |   
| Documentación API | OpenAPI 3.0 + swagger-ui-express |   
| Contenerización | Docker + Docker Compose |   
| Variables de entorno | dotenv |   
| CORS | cors |
